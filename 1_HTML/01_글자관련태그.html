글자 관련 기타 태그

strong 태그 i 태그 : 단순히 글자를 기울이는 태그
em 태그 : 글자를 강조하는 태그(웹 접근성)
mark 태그 : 형광펜 효과를 나타내는 태그
u 태그 : 밑줄을 긋는 태그
s 태그 : 취소선을 긋는 태그는 (비슷한 태그 del 태그가 존재)
del 태그
